# CSS Advanced Project

Welcome to the **CSS Advanced Project**, part of the [ALX Front-End Curriculum](https://www.alxafrica.com/).

This project focuses on converting a fully designed webpage (via Figma) into a **pixel-perfect responsive HTML & CSS implementation**.

---

## 📁 Project Directory

**Repository:** [alx_html_css](https://github.com/iamfikky/alx_html_css)  
**Project folder:** `css_advanced`  
**Main file:** [`index.html`](https://github.com/iamfikky/alx_html_css/blob/main/css_advanced/index.html)

## 🎯 Project Objectives

- Apply advanced CSS concepts
- Build a fully responsive webpage
- Reproduce a professional web design from a Figma file
- Structure HTML semantically and accessibly
- Use modern layout techniques (Flexbox, Grid)
